# Framework / Library Timers

## Topics

- Boost.Asio timers, such as `boost::asio::deadline_timer`
- Qt timers, such as `QTimer`
- libuv, used in Node.js and high-performance C++ backends
- POCO C++, ACE, or Chrono++

## Practice Ideas

- Build a GUI timer that updates a clock every second with Qt.
- Build an async socket with timeout using Boost.Asio.
- Implement retry logic with libuv or Boost timers.
