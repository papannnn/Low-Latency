# C++ Memory Model

These topics are the backbone of lock-free programming.

## Topics

- `std::atomic<T>` vs `volatile`
- Memory orderings:
  - `memory_order_relaxed`
  - `memory_order_acquire` / `memory_order_release`
  - `memory_order_acq_rel`
  - `memory_order_seq_cst`
- Data races vs race conditions

## Note

Try completing these within one weekend. Do not just memorize the concepts; try different experiments for better understanding and clarity.
